## 🎨 Foundations: Organic Ink
The **Organic Ink** design system is built around a grounded, calm, and typographically rich aesthetic. It moves away from generic SaaS interfaces toward something that feels intentional and tactile.

- **Light Mode:** Warm Parchment (`#F5F0E8`) & Ivory surfaces.
- **Dark Mode:** Soft Charcoal (`#181612`) & Deep Ink elevations.
- **Typography:**
  - `Outfit` for modern, clean UI.
  - `JetBrains Mono` for precise, technical data.
  - `Cormorant Garamond` for timeless, elegant headings.

---

## ✨ Wow Factors
Small details that make this project stand out from standard authentication systems:

- **Interactive Ink Background:** A generative canvas that creates subtle, reactive ink bleeds as you move your mouse.
- **Digital Signature Pad:** Authenticate your profile changes with a handwritten style signature rendered via HTML5 Canvas.
- **Dynamic Avatars:** UI-driven initials with deterministic organic gradients based on user identity.
- **Fluid Layouts:** Non-shifting button loaders and staggered entrance animations for a premium feel.

## 🏗 Architecture
Built for reliability and speed, utilizing a multi-database strategy:

| Component | Technology | Role |
| :--- | :--- | :--- |
| **Persistence** | MySQL | Relational data for secure account management. |
| **Profile** | MongoDB | Document store for flexible, nested user metadata. |
| **Sessions** | Redis | Ultra-fast token management and caching. |
| **Logic** | PHP 8.1+ | Robust, secure API endpoints with Bearer middleware. |
| **Client** | jQuery | Efficient AJAX-driven interaction and UX updates. |

## 🚀 Deployment
This project is fully optimized for **Vercel** using the latest `vercel-php` runtime.

```bash
# Clone the repository
git clone https://github.com/Kritika11052005/auth-system.git

# Configure environment
cp .env.example .env

# Deploy via Vercel CLI
vercel --prod
```
